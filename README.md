# WiFi Network Analyzer

A Python-based command line tool that scans nearby WiFi networks and analyzes their signal strength and security.

## Features

- Scan nearby WiFi networks
- Display signal strength and quality
- Detect network security (Open, WEP, WPA, WPA2/WPA3)
- Identify crowded WiFi channels
- Detect possible rogue access points
- Live refresh scanning
- Clean terminal dashboard using Rich library

## Technologies Used

- Python
- PyWiFi
- Rich (for terminal UI)
